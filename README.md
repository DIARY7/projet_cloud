"# projet_cloud" 

## Lancement docker

Pour initialiser les applications:
```bash
    docker compose up --build -d
```

Pour lancer .net:
```bash
    docker exec -it dotnet_sdk /bin/bash
    cd  userboard
    dotnet run
```

Les liens:
    - API .net:
        http://localhost:5000/
    - API java:
        http://localhost:8080/ 
    - REACT
        http://localhost:3000/
    - GitHub:
